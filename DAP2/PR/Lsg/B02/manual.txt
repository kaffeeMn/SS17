======================================================================
======================= Sortierung.java Manual =======================
======================================================================
Die Eingabe muss nach dem Schema:
    java Sortierung <ZAHL>
                           ^-- (defaults to [insert auf])
    java Sortierung <ZAHL> [insert|merge] 
                                          ^-- (defaults to [auf])
    java Sortierung <ZAHL> [insert|merge [auf|ab|rand]]
erfolgen, ausserdem muss <Zahl> aus dem Wertebereich INTEGER kommen.
